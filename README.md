# Muhammad Anas Khan — Portfolio

Personal portfolio website built with Next.js 14, Tailwind CSS, and Framer Motion.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build for production

```bash
npm run build
npm start
```

## Deploy to Vercel

Push to GitHub and import the repo in [vercel.com](https://vercel.com). No env vars required.

## CV

Place your CV PDF at `public/Anas_Khan_CV.pdf` to enable the Download CV button.

## Stack

- **Framework** — Next.js 14 (App Router)
- **Styling** — Tailwind CSS
- **Animations** — Framer Motion
- **Icons** — Lucide React
- **Fonts** — JetBrains Mono, Inter (Google Fonts)
